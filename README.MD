# IO_MESH_YMP
## Introduction
`io_mesh_ymp` is an addon that imports Yuke's **YMP** files (.YOBJ) into Blender. Currently, PS2 models are only supported.
## Checklist
* Imports Vertex Positions (Y)
* Imports Vertex Normals (Y)
* Imports Vertex Colours (N)
* Skins vertices (Y)
* UV supported (Y)
* Axis corrected (Y)
* Armatures and bounding spheres (Y)
* proper subobject and parent object parsing (Y)
* exports back (N)
